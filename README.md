# JAM-Productions Portfolio & Blog

This is the repository for the JAM-Productions Portfolio & Blog. It is built using Astro, a static site generator that allows for the use of React components.

The members and the projects are defined in its own JSON file and are loaded into the site using the Astro. Also, the blog posts are loaded from the markdown files in the `src/content/blog` directory.

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |
